# Star Wars: The Release Order

A minimalist, single-page web experience visualizing the essential Star Wars saga from 1977 to 2026.

## Overview

This curated timeline presents the Star Wars universe following the **Lucasfilm/Filoni narrative thread**, focusing on entries that contribute meaningfully to the overarching story while minimizing filler content.

## Features

- **Animated Starfield Background** - Canvas-based parallax scrolling with twinkling stars and shooting stars
- **Glassmorphic UI** - Modern frosted glass panels with subtle backdrop blur effects
- **Fully Responsive** - Optimized viewing experience across desktop, tablet, and mobile devices
- **Smooth Interactions** - Scroll-to-top button and fluid animations throughout

## Content Philosophy

**Included:**
- Original trilogy (Episodes IV-VI)
- Prequel trilogy (Episodes I-III)
- The Clone Wars (curated episodes only)
- Essential animated series (Rebels, Bad Batch, Tales series)
- Standalone films (Rogue One, Solo)
- Post-Empire era shows (Mandalorian, Ahsoka, Andor, etc.)
- Canon video games (Jedi: Fallen Order, Jedi: Survivor)

**Excluded:**
- Episodes VII, VIII, IX (Sequel trilogy)
- *The Acolyte*
- Narrative filler episodes

## Media Categories

The timeline encompasses four types of content:

- **Feature Films** - Theatrical releases
- **Live Action** - Disney+ series
- **Animated** - TV series and limited series
- **Video Games** - Story-driven canon games

## Technical Stack

- Pure HTML5, CSS3, and Vanilla JavaScript
- Canvas API for starfield animation
- CSS Grid and Flexbox for responsive layouts
- Custom fonts from Google Fonts

## Updates

This list is maintained and updated **quarterly** to include new releases and refinements to the curated episode selections.
